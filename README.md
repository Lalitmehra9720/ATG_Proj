## Getting Started

After cloning the repo:

```bash
npm i
npm run dev
```


## Env variables needed
You want to environment variable inside **_src/.env_**
```bash
MONGO_URI: YOUR_MONGODB_URL
JWT_SECRET: YOUR_JWT_SECRET 
```

This urls are used for authentication routes inside **_src/app/api/auth/signup_** and  **_src/app/api/auth/login_**

## All homepage code 
Homepage code is inside **_src/app/(public)_**

## All auth pages are inside
Homepage code is inside **_src/app/(auth)_**

## All dashboard pages are inside
Dashboard code is inside **_src/app/dashboard_**

## All auth apis are inside
Auth api code is inside **_src/app/api/auth_**
